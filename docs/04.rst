自動操縦でドローンを飛ばしてみよう
==================================================

これから、ドローンを自動操縦するプログラムのプログラムを作成します

autoPilotメソッドの追加
-------------------------------------------

以下のコードを「controller.js」追加します。

.. code-block:: js

  // 自動操縦のキュー
  function autoPilot() {

      temporal.queue([
        {
          delay: 5000,
          task: function () {
            d.takeOff();
            d..flatTrim();
          }
        },
        {
          delay: 3000,
          task: function () {
            d.forward({steps: 12});
          }
        },
        {
          delay: 5000,
          task: function () {
            d.land();
          }
        },
        {
          delay: 5000,
          task: function () {
            d.clear();
            d.exit(0);
          }
        }
      ]);
  }

次に以下のコードを追加します。

.. code-block:: js

  // 自動操縦
  else if (key.name === 'b') {
    console.log('autoPilot');
    autoPilot();
  }

このコードはキーボードからの入力を受けるため process.stdin.on('keypress', function (ch, key) {} の中に追加してください。

追加したら、下記コマンドをコマンドプロンプトから実行し、キーボードの”b”を押してみましょう。

.. code-block:: none

  node controller.js
